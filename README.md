# powershell-lib

Collection of old PowerShell functions.

Includes:

- Create-PSCredential
- Alter-FileAndFolderPermission
- Copy-Acl
- Get-UpTime
- Get-ADComputerLastLogonTimestamp
- Get-OSProductKey
- Get-RemoteOSProductKey
- Set-DefaultPrinter
- Get-SystemInfo

Would import as a PowerShell module.

```PowerShell
Import-Module .\lib.ps1
```
